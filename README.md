# 校园二手交易平台

## 项目简介
校园二手交易平台是基于Java语言，Springboot框架， Mysql数据库，Html/JavaScript/Css技术的高校二手交易平台。系统包含：前端和后端项目。角色分为：普通学生、管理员、 普通学生可以登录发布自己的二手产品，可以购买其他学生发布的二手产品；管理员对商品、用户等进行管理、适合作为毕业设计、大作业、课程设计。
